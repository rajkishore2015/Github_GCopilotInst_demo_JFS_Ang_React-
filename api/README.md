# UserApp API (Spring Boot)

Spring Boot 3 (JDK 21) microservice API.

## Prerequisites
- Java 21 (JDK)
- Maven 3.9+

## Run
```bash
mvn spring-boot:run
```

## Tests & Coverage
```bash
mvn verify
```
Coverage reports are generated by JaCoCo under target/site/jacoco.

## Configuration
Default configuration is in src/main/resources/application.yml. Override via environment variables.
